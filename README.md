# Resolution AI

AI-powered resolution memory system that turns customer support activity into product roadmap intelligence.

## 🎯 The Problem

- 70% of critical support knowledge is never documented
- New CSMs take 3-6 months to ramp up  
- PMs spend 10+ hours/week manually analyzing tickets
- When senior CSMs leave, knowledge walks out the door

## ✨ The Solution

**For CSMs:** Log issues, get instant step-by-step solutions. Vote helpful/not helpful - system learns.

**For PMs:** Dashboard auto-populates from CS activity. Knowledge gaps, recurring patterns, client impact - all without reading tickets.

## 🛠️ Tech Stack

- React 18
- Claude API (Anthropic)
- LocalStorage (demo) - production would use PostgreSQL  
- Vercel deployment

## 📊 Features

### CSM Workspace
- AI-powered solution matching
- Step-by-step resolution instructions
- Reddit-style voting system
- CSM attribution tracking

### PM Analytics  
- Knowledge gap detection
- AI-powered root cause analysis
- Recurring pattern identification
- Client impact prioritization
- Top contributors leaderboard

### Knowledge Base
- 13 documented resolutions
- Category filtering
- Smart sorting
- Full context modals
